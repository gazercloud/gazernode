# gazernode
Monitoring node. Single executable file. Works as Windows Service. UI-client.

# Unit types

## File system sensors
- File Content
- File Size

## General sensors
- CGI
- CGI Key=Value
- HHGTTG

## Network sensors
- Ping
- TCP Connect

## Serial Port sensors
- Serial Port Key=Value

## Windows sensors
- Windows Memory
- Windows Process
- Windows Networks
